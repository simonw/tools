# Syntaqlite Playground

Experiment with SQLite SQL parsing, formatting, validation, and tokenization directly in your browser using the syntaqlite parser. This interactive playground provides four main tools: a code formatter that restructures SQL with customizable options, a parser that generates abstract syntax trees, a validator that checks queries against a defined schema, and a tokenizer that breaks SQL into individual tokens with type classification. All processing happens client-side via Pyodide, which runs Python code in WebAssembly without requiring any server infrastructure.
