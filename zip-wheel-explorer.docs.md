Browse and view the contents of Python package files (.whl, .zip, and .tar.gz archives) directly in your browser by entering a PyPI package name or a direct file URL. The tool automatically downloads the package, extracts all files, and displays them in an interactive list with syntax highlighting and line-by-line navigation, allowing you to link directly to specific files and lines using URL fragments for easy sharing.
