Web application for viewing Mozilla Bugzilla bug reports with full details including comments, change history, and related bugs. The viewer fetches bug data from the Bugzilla REST API and displays it in a chronological timeline format, with visual indicators for important changes like status updates and dependency modifications. Users can search by bug ID or paste a Bugzilla URL directly, with the application automatically formatting author names and URLs while preserving privacy by hiding email addresses.
