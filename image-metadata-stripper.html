Image Metadata Stripper

Remove EXIF data, GPS location, camera info, and other metadata from images. All processing happens locally in your browser.

How it works

  • JPEG: Lossless removal using piexifjs - strips EXIF without re-encoding
  • PNG: Lossless chunk filtering - removes metadata chunks while preserving image data
  • Other formats: Re-encoded via canvas (may affect quality)
