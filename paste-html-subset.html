Rich Paste to HTML Subset

Supported HTML elements

  • p - Paragraphs
  • strong - Bold text
  • em - Italic text
  • blockquote - Quoted text
  • code - Code snippets
  • a - Links (href attribute is preserved)
  • h1-h6 - Headings
  • ul - Unordered lists
  • ol - Ordered lists
  • li - List items
  • dl - Definition lists
  • dt - Definition terms
  • dd - Definition descriptions
