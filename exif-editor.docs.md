View and edit image metadata using the full ExifTool running in WebAssembly. Displays all EXIF, IPTC, XMP, and other metadata in a categorized view with raw JSON output. Edit common fields like title, author, copyright, description, and keywords, then download the modified image. Supports JPEG, PNG, TIFF, WebP, HEIC, and many other formats. Powered by @uswriting/exiftool which runs ExifTool 13.42 entirely in the browser with no server uploads.
