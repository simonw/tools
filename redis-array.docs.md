# Redis Array Playground

Explore Redis array commands in an interactive web-based environment compiled to WebAssembly. This playground lets you execute AR* commands against an in-memory Redis instance, visualize keyspace contents, and inspect command documentation with live syntax previews. The implementation uses the sparse array data structure from the antirez:array branch, exposing commands like ARSET, ARGET, ARGREP, and others alongside a console that displays all executed commands and their responses.
